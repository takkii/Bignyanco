## Bignyanco is neovim plugins. 

環境 ( Enviroments )

```
neovim
python3
deoplete.nvim
$XDG_CONFIG_HOME=~/.config/nvim
```

## deinでプラグインを管理しているとき

> call dein#add('takkii/Bignyanco')

init.vimに追加します。

## deopleteを動作させるために、

>:UpdateRemotePlugins

>:CheckHealth

コマンドを入力して、結果を確認します。

エラーがあればメッセージをみて対処します。

※ deopleteのスタートアップ記述は書いたよね。
